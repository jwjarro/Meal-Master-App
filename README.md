Welcome!

When you first start the app (or if you'd like to reset for the next semester):

  1. Open Settings
  2. Enter the following information:
       i. Total number of weeks
      ii. The current week (the first week should be 1)
     iii. The total amount of dining dollars
      iv. The number of swipes alloted per week
  3. Hit Save
  4. Hit Progress Week
  5. Enjoy!
